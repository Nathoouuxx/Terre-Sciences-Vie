Si tu es débutant et que tu dois créer un site web avec un minimum de code, voici une solution étape par étape très détaillée, même pour les choses qui peuvent sembler évidentes. Je vais te proposer des outils gratuits et te guider pour un site en rapport avec ta filière (L2 BCPP = Biologie Cellulaire, Physiologie et Neurosciences). --- ### **Étape 1 : Choisir une idée de site en rapport avec la BCPP** Quelques idées simples : 1. **Site sur les neurones** : Expliquer leur fonctionnement avec des images. 2. **Blog sur les découvertes en neurosciences** : Articles simples sur des recherches récentes. 3. **Glossaire de biologie cellulaire** : Définitions des termes importants. 4. **Page de présentation d’un mécanisme physiologique** (ex : la synapse). ➡ **Choix proposé** : Un site simple sur **"Les Neurones et la Communication Synaptique"** (tu peux trouver des infos sur Wikipédia ou ton cours). --- ### **Étape 2 : Choisir un outil pour coder et héberger gratuitement** Pour un vrai code (même minimal), utilise : - **GitHub Pages** (hébergement gratuit + tu peux écrire du HTML/CSS). - **Replit** (éditeur en ligne + hébergement temporaire gratuit). ➡ **Choix proposé** : **GitHub Pages** (car c’est pris au sérieux dans le milieu académique). --- ### **Étape 3 : Créer un compte GitHub** 1. Va sur [github.com](https://github.com) 2. Clique sur **"Sign up"** 3. Remplis : email, mot de passe, nom d’utilisateur (ex: **JeanDupontBCPP**). 4. Valide ton email. --- ### **Étape 4 : Créer un dépôt (repository)** 1. Une fois connecté, clique sur **"+"** en haut à droite → **"New repository"**. 2. Donne un nom : **"neurones-site"** (sans espaces). 3. Coches **"Public"**. 4. Coche **"Add a README file"**. 5. Clique sur **"Create repository"**. --- ### **Étape 5 : Ajouter des fichiers HTML et CSS** 1. Dans ton dépôt, clique sur **"Add file"** → **"Create new file"**. 2. Nomme le fichier **"index.html"** (obligatoire pour la page d’accueil). 3. Copie ce code minimal : ```html Les Neurones - BCPP

La Communication Neuronale

En neurosciences, les neurones communiquent via des synapses.

Source : Wikipédia

``` 4. Clique sur **"Commit changes"** (valider). --- ### **Étape 6 : Activer GitHub Pages** 1. Dans ton dépôt, va dans **"Settings"** (en haut à droite). 2. Dans le menu de gauche, clique sur **"Pages"**. 3. Sous **"Branch"**, choisis **"main"** et clique sur **"Save"**. 4. Attends 2 minutes, puis rafraîchis la page. Un message apparaîtra avec le lien de ton site (ex: `https://JeanDupontBCPP.github.io/neurones-site`). ➡ **Félicitations ! Ton site est en ligne.** --- ### **Étape 7 : Personnaliser (optionnel mais recommandé)** - Pour ajouter un fichier CSS séparé : 1. Crée un fichier **"style.css"** dans le même dépôt. 2. Modifie ton HTML pour ajouter : ```html ``` - Pour ajouter plus de contenu BCPP : - Va sur [PubMed](https://pubmed.ncbi.nlm.nih.gov/) ou [Wikipédia](https://www.wikipedia.org/) pour trouver des infos sur les neurones. - Ajoute des sections comme : ```html

Les Différents Types de Neurones

  • Neurones sensoriels
  • Neurones moteurs
``` --- ### **Étape 8 : Montrer ton code (pour la validation)** - Partage le lien GitHub du dépôt (ex: `https://github.com/JeanDupontBCPP/neurones-site`). - Et le lien du site en ligne (ex: `https://JeanDupontBCPP.github.io/neurones-site`). --- ### **Autres options si GitHub est trop compliqué** 1. **Replit** : - Va sur [replit.com](https://replit.com), crée un compte. - Crée un nouveau projet **"HTML/CSS"**. - Copie le code HTML ci-dessus. - Clique sur **"Run"** pour prévisualiser. - Partage le lien Replit (ex: `https://replit.com/@JeanDupontBCPP/neurones`). 2. **Netlify** (hébergement gratuit plus simple) : - Télécharge ton fichier HTML. - Va sur [netlify.com](https://www.netlify.com), fais un glisser-déposer du fichier. --- ### **Résumé des liens importants** - **GitHub Pages** : [https://pages.github.com](https://pages.github.com) - **Exemple de site fini** : [https://jeandupontbcpp.github.io/neurones-site](https://jeandupontbcpp.github.io/neurones-site) (remplace par ton nom d’utilisateur).
